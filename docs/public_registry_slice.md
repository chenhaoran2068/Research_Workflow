This file is a public-safe slice of the current workflow registry structure.

## Registry Interpretation Rule

The registry is treated as:

- human-authored design-time governance truth
- something that runtime and audit consume
- something runtime is not allowed to invent on its own

That means:

- coordinate membership is authored
- coordinate membership is not inferred from logs alone
