## Purpose

This file maps the main public workflow-pack claims to the frozen evidence
families that support them.

It keeps claim language compact and bounded.

## Claim-Safety Rule

These evidence families support workflow-pack claims.

They do **not** automatically support:

- full paper conclusions
- universal external generalization
- claims beyond the bounded frozen experiment surfaces summarized here
